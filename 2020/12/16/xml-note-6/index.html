XML学习笔记#5:XQuery

发表于2020-12-16更新于2020-12-16阅读次数

XQuery(An XML Query Language) 是一种专门用于XML的半结构化数据查询语言。这是一篇XQuery的笔记。

概述

XQuery的功能与XSLT类似，但是写起来较为直观简便。

自定义函数

XQuery允许我们自定义函数，其基本框架为 declare function 命名空间:函数名 (参数列表) as 返回值类型 {函数体}
